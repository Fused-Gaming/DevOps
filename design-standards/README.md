# VLN Design Standards

> **Official design system and engineering standards for VLN products**

Built with Docusaurus. Visit **https://design.vln.gg** when deployed.

## Quick Start

```bash
npm install
npm start       # Development server
npm run build   # Production build
```

## What's Inside

- **Design System**: Colors, typography, components, spacing
- **Tools**: Open source design tools (Penpot, Inkscape, GIMP)
- **Workflows**: ASCII design, mockup workflow, prototyping
- **Responsive**: Breakpoints, layouts, testing strategies
- **Branding**: Logo, voice/tone, assets

## Key Features

- WCAG AAA accessible color system
- Mobile-first responsive design
- 100% open source tools ($0/month vs. $75+/month for commercial)
- Integration with Next.js, Tailwind CSS, Framer Motion
- Comprehensive component documentation

## Tech Stack

- Docusaurus 3.9.2
- React 19
- TypeScript
- MDX
