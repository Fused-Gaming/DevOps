# Brand Assets

VLN brand assets and graphics library.

## Available Assets

Located in `/public/`:
- `logo.png` - Main logo (512×512px)
- `og-image.png` - Social sharing image (1200×630px)
- `twitter-card.png` - Twitter card (1200×675px)
- `favicon.ico` - Browser favicon
- `apple-touch-icon.png` - iOS icon (180×180px)

## Social Media Specs

| Platform | Dimensions | File |
|----------|------------|------|
| **Open Graph** | 1200×630px | og-image.png |
| **Twitter** | 1200×675px | twitter-card.png |
| **LinkedIn** | 1200×627px | Use og-image.png |

## Export Guidelines

- Use PNG for logos with transparency
- Use WebP for photos (better compression)
- Optimize all images before committing
- Use Next.js Image component in code

See [Design Tools](/tools/design-tools) for export workflows.
