# Logo Usage

Guidelines for using the VLN logo across products.

## Logo Files

Available in `/public/`:
- `logo.png` - 512×512px PNG
- `favicon.ico` - Browser favicon
- `apple-touch-icon.png` - 180×180px iOS icon

## Usage Guidelines

- Maintain clear space around logo
- Don't distort or stretch
- Use on dark backgrounds
- Minimum size: 32px height

## Download

Logo files are available in the [DevOps repository](https://github.com/Fused-Gaming/DevOps/tree/main/public).
