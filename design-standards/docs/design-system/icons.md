# Icons

VLN uses Lucide React for a consistent, beautiful icon system.

## Usage

```tsx
import { Rocket, Heart, Star } from 'lucide-react';

<Rocket className="text-vln-sage" size={24} />
<Heart className="text-red-500" size={20} />
<Star className="text-vln-amber" size={32} />
```

## Browse Icons

Visit [lucide.dev](https://lucide.dev/icons) to browse 1000+ icons.
